== Description ==

WeldPress is a mobile platform that helps you quickly develop high-quality apps. WeldPress is made up of complementary features that you can mix-and-match to fit your needs.
Can manage multiple applications, plus you can customize the applications.

== Installation ==

Installation

IN THE ADMIN

1. Search for “WeldPress”
2. Click “install”
3. Upon success notification click “Activate Plugin”
4. Enjoy!

MANUAL INSTALL

1. Click “Download Version 4.9.1”
2. Go to your WordPress site, to “Plugins > Add New”
3. Click on “Upload”
4. Click on the “Choose File” button and find the zip you just downloaded
5. Click on the “Install Now” button
6. Upon success message, click “Activate Plugin”
6. Enjoy!


You will find 'WeldPress' menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin homepage](https://weldpress.org/gettingstarted/).
